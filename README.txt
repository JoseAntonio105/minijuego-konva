Idea de juego (desarrollándose todavía): Juego de gestión de comercio entre ciudades simple
Nombre provisional: TradeCity

El juego tendrá un diseño MVC escalable.

Las carpetas definidas son las siguientes, y se explica la utilidad de cada una:

- assets (Almacenar los archivos multimedia usados en el juego)
- config (Configuración de datos estáticos del juego)
- controller (Controladores)
- core (Núcleo de la aplicación)
- model (Lógica de datos)
- utils (Utilidades del proyecto)
- view (Vista del juego)
